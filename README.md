# ProyectoFinalM12
Este repositorio esta destinado a un proyecto de final de curso de unos alumnos del isntituto Joaquim Mir Vilanova i la Geltru,
Bienvenido al repositorio del proyecto en desarrollo de nuestra aplicación social orientada a los entusiastas de la conducción,
como parte de nuestro trabajo en el instituto. Esta aplicación, desarrollada con tecnologías como React Native para el frontend y
PHP con Laravel para el backend, tiene como objetivo crear una comunidad en línea para aquellos que comparten una pasión por la conducción.
